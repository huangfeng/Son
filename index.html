Son by twfarland

An embedded css authoring DSL in Coffeescript/Javascript, with Sass-like abilities

Want to reduce repetition and create better abstractions in your css? Don't want to install and learn a dedicated solution such as Sass / Less / Stylus for this? Just write your css as coffeescript/javascript and compile it with Son. Son brings all the power of javascript to css, and maps more closely to underlying css semantics than related solutions, e.g: by preserving the ordering of property declarations.

Install

npm install son
